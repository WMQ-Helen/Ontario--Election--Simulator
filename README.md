# Ontario--Election--Simulator

This project compares two voting systems: First-Past-the-Post (FPTP) and Alternative Vote (AV). The goal is to examine whether changing the voting system affects riding winners and the total number of seats won by each party.

## Data Source

The project uses simulated voter data generated in Python. The dataset contains 2,500 voters across some Ontario locations, and each voter ranks four political parties: PC, Liberal, NDP, and Green.

## Data Preparation

Because the data was generated by the program, extensive cleaning was not required. The main preparation steps were separating voters by riding and converting each preference ranking from a text string into a Python list.

## Analysis Method

Under FPTP, the party with the most first-choice votes wins the riding. Under AV, the party with the fewest votes is eliminated, and its votes are transferred to each voter’s next preferred active party until one party receives more than 50% of the votes.

## Results

In this simulation, both voting systems produced the same seat distribution. The Liberal Party won 6 seats, while the PC Party and NDP each won 2 seats. The Green Party won no seats.

Although the final seat totals were the same, AV changed the counting process by transferring votes and requiring the final winner to receive majority support.
